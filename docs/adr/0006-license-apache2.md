# ADR-0006: Apache-2.0 license

- **Status**: Proposed
- **Date**: 2026-07-11

## Context
Goal: become a neutral standard that companies and agent vendors will adopt and write
adapters for.

## Decision
Apache-2.0 for the core and all first-party plugins.

## Consequences
Patent grant and corporate-friendliness maximize adapter contributions; copyleft
(AGPL) would deter exactly the vendors whose participation makes the platform a
standard. Trademark ("Prodeo Command Center") retained separately to protect the
name once relevant.

## Alternatives Considered
**MIT** (fine, but no patent grant). **AGPL** (protects against SaaS enclosure at
the cost of adoption; wrong trade for a would-be standard).
